What is Git?

Git is a Distributed Version Control System (DVCS) used to track code changes.

Key Features:

Open source
Platform independent
Handles large projects
Written in C
Released in 2005

Why Git is a Distributed Version Control System?

Every developer has a complete copy of the repository
Full history exists locally
No single point of failure

Benefits:
Offline commits
Faster operations
Better security
Easy recovery
